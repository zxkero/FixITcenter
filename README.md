# FixIT Center

Веб-сайт сервисного центра по ремонту цифровой техники.

## О проекте

FixIT Center — это многостраничный сайт сервисного центра.
Проект демонстрирует базовую структуру фронтенд-приложения:

- Главная страница
- Страница услуг
- Страница цен
- Страница «О компании»
- Контакты
- Модальное окно заявки
- Адаптивная навигация (бургер-меню)

---

## Используемые технологии

- HTML5
- CSS3
- JavaScript
- Git
